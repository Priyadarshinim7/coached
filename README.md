# Priyadarshini's Personal Website

A professional personal website for Priyadarshini, an Electronics and Communication Engineering student at NIE, Mysore.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI**: Clean, minimal, and elegant design using a blue-gray-white color palette
- **Professional Sections**: 
  - Header with name and tagline
  - Navigation menu
  - Home section with introduction
  - About Me section with bio
  - Skills section showcasing technical abilities
  - Resume download section
  - Contact information
- **Google Fonts**: Uses Poppins font family for modern typography
- **Smooth Animations**: Hover effects and transitions for better user experience

## File Structure

```
HTML_FE/
├── INDEX.HTML          # Main HTML file
├── STYLE.CSS           # CSS stylesheet
├── resume.pdf          # Sample resume file

```

## Setup Instructions

1. Download or clone the project files
2. Open `INDEX.HTML` in a web browser
3. The website should display correctly with all styling applied

## Customization

To customize the website for different use:

1. **Update Personal Information**: Modify the content in `INDEX.HTML`
2. **Change Colors**: Update the CSS variables in `STYLE.CSS`
3. **Add Sections**: Extend the HTML structure and add corresponding CSS
4. **Update Resume**: Replace `resume.pdf` with actual resume file
